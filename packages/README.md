# 多人端共享包

本目录承载多人实时系统的共享代码，供未来 `apps/server`、队长端和观众端复用。

## 当前包

| 包 | 职责 |
| --- | --- |
| `shared` | 角色、Command、Event、规则版本、状态 schema 版本和基础协议校验。 |
| `rules` | 服务端权威状态的最小运行时骨架，包含状态版本、幂等、队长队伍边界和事件追加。 |

## 当前边界

- 当前阶段只建立 M1 骨架，不迁移现有金币商店和海克斯完整规则。
- 现有裁判端仍使用根目录 `src/` 和 `apps/multiplayer/src/` 的浏览器全局实现。
- 后续迁移服务端时，新的 Command 入口应先通过 `packages/shared` 校验，再交由 `packages/rules` 和具体规则处理器执行。
