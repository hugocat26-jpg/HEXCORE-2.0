# HEXCORE2.0 文档目录

本目录保留项目协作和开发所需的源文档。编号 Markdown 文档位于当前目录，供任务钩子、开发计划和技术追踪直接引用。

## 目录约定

- 编号 Markdown：项目总览、架构、数据模型、执行流程、开发计划、规则与实施规范等源文档；过期的无金币、四轮迁移和历史设计方案已移除，不要求编号连续。
- `user-guides/`：面向用户、裁判或发布阅读的 Word 文档。
- `../output/pdf/`：由脚本生成的 PDF 发布资料。
- `../output/release/`：本地打包发布产物，不作为源文档维护。
- `../CHANGELOG.md`：版本更新记录；每次版本号变化和 GitHub 推送前必须同步维护。

## 维护说明

`scripts/task-loop-runner.js` 默认读取 `docs/06_开发计划.md`，因此不要随意移动编号 Markdown 文档。若需要调整目录结构，请同步更新任务钩子和回归测试中的路径。
