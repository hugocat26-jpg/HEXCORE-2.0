## 当前执行重点

当前排队项以 [planning/当前待开发计划.md](planning/当前待开发计划.md) 为准。默认任务钩子已读取新计划文档：

```powershell
npm run post-task -- --status=complete
```

旧命令仍可显式指定本文作为兼容入口：

```powershell
npm run post-task -- --doc=docs/06_开发计划.md --status=complete
```
